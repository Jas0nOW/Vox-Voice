
# Scope (Voice-System)

## In Scope ✅
- Mic Toggle via Right-Ctrl + VAD (on/off)
- STT lokal (faster-whisper)
- Routing: insert | stdout | cli:gemini | cli:ollama
- Insert: active (typing→clipboard), clipboard, off
- Safe Mode
- Permissions (ask/allow/deny) persistent
- Audit Logging (jsonl) + Privacy Toggle
- Optional Orb (tkinter) + CLI UI

## Out of Scope ❌
- Wake-Word / Always-on Listening
- Global hotkeys unter Wayland
- Voller WANDA Orchestrator / Memory / Agent-Permissions global
